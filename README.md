# Database Locking Demo
Contains demo code to lock database

## Dependencies

- git
- [composer](https://getcomposer.org/download/)
- docker
- [docker-compose](https://docs.docker.com/compose/)

## Install

```bash
$ git clone git@github.com:tienvx/db-locking-demo.git
$ cd db-locking-demo
$ composer install
$ cp .env.dist .env
$ docker-compose up
```
